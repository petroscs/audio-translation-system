Run translator/listener app on Windows (desktop)
----------------------------
First start the backend so the app can reach the API:
  > cd audio-translation-system\backend
  > dotnet run --project Backend.Api

Then run the app with the API URL pointing at this machine:
  > cd audio-translation-system\mobile\translator_app
  > flutter run -d windows --dart-define=API_BASE_URL=http://127.0.0.1:5000

  > cd audio-translation-system\mobile\listener_app
  > flutter run -d windows --dart-define=API_BASE_URL=http://127.0.0.1:5000

If you see "The remote computer refused the network connection", the backend is not running or the URL/port is wrong.
